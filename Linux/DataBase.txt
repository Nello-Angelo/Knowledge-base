su - postgres (зайти под пользователем с полными правами, в домашней папке есть все что нужно для настройки) psql -U postgres (перейти в консоль бд, если подключились к базе через форвард то добавляем -h localhost -p number-of-port) 
pg_dump имя бд > 12/backups/dump.sql (создаем backup ввиде sql скрипта под юзером postgres, если подключились к базе через форвард то добавляем -h localhost -p number-of-port )
psql -1 имя бд < 12/backups/dump.sql (команда для восстановления backup под юзером postgres, и если будет ошибка то откатить все к нулю, если подключились к базе через форвард то добавляем -h localhost -p number-of-port )
